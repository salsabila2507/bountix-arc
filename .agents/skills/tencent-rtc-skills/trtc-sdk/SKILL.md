# TRTC SDK Skills

## Overview

This directory contains skills related to **TRTC SDK** (Tencent Real-Time Communication SDK) — the low-level real-time audio/video SDK that powers Tencent RTC products.

These skills are for developers who work directly with the TRTC SDK rather than the higher-level UIKit products (TUIRoomKit, TUICallKit, TUILiveKit, TUIKit).
